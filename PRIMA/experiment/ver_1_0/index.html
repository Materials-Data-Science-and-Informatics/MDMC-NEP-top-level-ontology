Experimentc

URI https://purls.helmholtz-metadaten.de/prima/core#Experiment
Description

Experiment is an identifiable and reproducible activity with a clear start time and end time, which may include a set of one or more sample component syntheses, sample preparations and/or measurements, performed by one or more research users.
Super-classes prov:Activityc
Restrictions hasMeasurementop some Measurementc

ResearchUserc

URI https://purls.helmholtz-metadaten.de/prima/core#ResearchUser
Description

Research user is an agent or person, usually member of a project, who conducts any part of the experiment(s), or performs any of the steps of the data analysis lifecycle during the course of one or more studies.
Super-classes prov:Agentc
In range of https://raw.githubusercontent.com/Materials-Data-Science-and-Informatics/MDMC-NEP-top-level-ontology/master/mdmc-nep-top-level-ontology.owl/wasOperatedByop

Experimentc

URI https://purls.helmholtz-metadaten.de/prima/dal#Experiment
Description

Raw data is research data which is the primary output of a measurement, collected by one or more research users using an instrument, before any subsequent data processing.
Super-classes prov:Entityc
Restrictions prov:wasGeneratedByop some Measurementc

Equipmentc

URI https://purls.helmholtz-metadaten.de/prima/experiment#Equipment

URI https://purls.helmholtz-metadaten.de/prima/experiment#Laboratory

Measurementc

URI https://purls.helmholtz-metadaten.de/prima/experiment#Measurement
Description

Measurement is identifiable and reproducible activity, performed by one or more research users, who generate a single self-consistent unit of raw data about a sample or a set of them using an instrument under constant or varying controlled conditions, depending on the particular research context.
Super-classes prov:Activityc
Restrictions prov:wasAssociatedWithop exactly 1 Instrumentc

MeasurementTechniquec

URI https://purls.helmholtz-metadaten.de/prima/experiment#MeasurementTechnique

Measurement technique is techique or technology corresponding to the method used during a measurement to collect raw data about a sample or a set of them with an instrument.

URI https://purls.helmholtz-metadaten.de/prima/experiment#Sample

SampleComponentc

URI https://purls.helmholtz-metadaten.de/prima/experiment#SampleComponent
Description

Sample component is identifiable entity (typically a piece of material) with distinctive properties (structural, chemical, dimensional, functional and others), which is fabricated during the sample component fabrication and is used during the sample preparation to produce a sample.
