Data Acquisition c

IRI https://purls.helmholtz-metadaten.de/prima/core#DataAcquisition
Description

Set of actions carried out by one or more research users, performed on a system or a set of them to generate a single self-consistent unit of raw data using a technique, an instrument and other equipment under constant or varying controlled conditions described by setting(s), depending on the particular research context.
Sub Class Of https://w3id.org/pmd/co/Process c

  • https://w3id.org/pmd/co/output op some

Data Analysis Lifecycle c

IRI https://purls.helmholtz-metadaten.de/prima/core#DataAnalysisLifecycle
Description

Set of processes carried out by one or more research users, performed on research data using one or more techniques and/or research software in order to produce synthesized knowledge (e.g., to detect patterns, determine relationships, develop explanations, test hypotheses and/or prove theories) and to eventually suggest the conclusions of a study.
Sub Class Of https://w3id.org/pmd/co/Process c
In Domain Of uses research software op
In Range Of has data analysis life cycle op
Restriction

  • has technique op some
  • uses research software op some
  • https://w3id.org/pmd/co/input op some
  • https://w3id.org/pmd/co/output op some

Equipment c

IRI https://purls.helmholtz-metadaten.de/prima/core#Equipment
Description

Any kind of physical or virtual item, device, machine or other tools used to perform one or more Fabrication(s), Sample Preparation(s), Model Preparation(s), Data Acquisition(s) and/or any of the processes included in the Data Analysis Lifecycle.
Sub Class Of Agent c
In Range Of
  • operates op
  • uses equipment op
Restriction has setting op some
Super Class Of Instrument c

Sub Class Of Equipment c

Project c

IRI https://purls.helmholtz-metadaten.de/prima/core#Project

Project is an enterprise (potentially individual but typically collaborative) of one or more research users, planned to perform one or more studies.
Sub Class Of Entity c
In Domain Of has study op
Restriction has project member op some

Research Data c

IRI https://purls.helmholtz-metadaten.de/prima/core#ResearchData
Description

Data collected, created or examined by one or more research users to be analyzed or considered as a basis for reasoning, discussion or calculation in a research context, with the purpose of generating, verifying and validating original scientific claims that support the answer to some specific research question (i.e., conclusions).
Sub Class Of https://w3id.org/pmd/co/DigitalEntity c
Restriction was attributed to op some

Research Software c

IRI https://purls.helmholtz-metadaten.de/prima/core#ResearchSoftware
Description

Research software is software used to generate, process, analyse or access research data during any of the processes included in the data analysis lifecycle (possibly including data rendering, visualization, plotting).
Sub Class Of Software Agent c
In Range Of uses research software op
Restriction has setting op some

Research User c

IRI https://purls.helmholtz-metadaten.de/prima/core#ResearchUser
Description

Research user is an agent or person, usually member of a project, who conducts any part of the experiment(s), or performs any of the steps of the data analysis lifecycle during the course of one or more studies.
Sub Class Of Agent c
In Domain Of operates op
Restriction
  • operates op some
  • performs agent role op some

Setting c

IRI https://purls.helmholtz-metadaten.de/prima/core#Setting
Description

Set of configuration parameters which may be involved, for example, in a data acquisition (e.g., settings of the instrument), in any of the processes included in the data analysis lifecycle (e.g., settings of the research software), or to describe a model (e.g., by specifying the type of solver used).
Sub Class Of http://nfdi.fiz-karlsruhe.de/ontology/Specification c
In Range Of has setting op

Study c

IRI https://purls.helmholtz-metadaten.de/prima/core#Study
Description

Study is a set of one or more experiments and corresponding data analysis lifecycle performed by one or more research users who are part of the same project.
Sub Class Of Entity c
In Range Of has study op
Restriction
  • was attributed to op some
  • was attributed to op some
  • has data acquisition op some
  • has data analysis life cycle op some

System c

IRI https://purls.helmholtz-metadaten.de/prima/core#System
Description

Physical or digital entity or set of entities with distinctive properties (structural, chemical, dimensional, functional or others) which is the subject of one or more actions or investigations.
Sub Class Of Entity c

Technique c

IRI https://purls.helmholtz-metadaten.de/prima/core#Technique
Description

Any experimental, theoretical or computational method used during data acquisition or during any of the processes included in the data analysis lifecycle to acquire, process or analyze research data about a system or a set of them with an instrument.
Sub Class Of http://nfdi.fiz-karlsruhe.de/ontology/Specification c
