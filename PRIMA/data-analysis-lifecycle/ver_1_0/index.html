DataAnalysisLifecyclec

URI https://purls.helmholtz-metadaten.de/prima/core#DataAnalysisLifecycle
Description

Data analysis lifecycle is a set of processes carried out by one or more research users, who systematically perform actions and apply methods on research data in order to produce synthesized knowledge to, e.g., detect pattern, determine relationships, develop explanations, test hypotheses, prove theories, suggest the conclusions of the study.
Super-classes prov:Activityc

terms:hasPartop some DataProcessingc

ResearchDatac

URI https://purls.helmholtz-metadaten.de/prima/core#ResearchData
Description

Research data is the data collected, created or examined by one or more research users to be analysed or considered as a basis for reasoning, discussion or calculation in a research context, with the purpose of generating, verifying and validating original scientific claims that support the answer to some specific research question (i.e., conclusions)
Super-classes prov:Entityc
Restrictions prov:wasAttributedToop some core:ResearchUserc

ResearchUserc

URI https://purls.helmholtz-metadaten.de/prima/core#ResearchUser
Description

Research user is an agent or person, usually member of a project, who conducts any part of the experiment(s), or performs any of the steps of the data analysis lifecycle during the course of one or more studies.
Super-classes prov:Agentc

AnalysedDatac

URI https://purls.helmholtz-metadaten.de/prima/dal#AnalysedData
Description

Analysed data is research data which is the primary output of any kind of data analysis performed on research data, typically on processed data, by one or more research users, possibly using research software.

DataAnalysisc

URI https://purls.helmholtz-metadaten.de/prima/dal#DataAnalysis
Description

Data analysis is data treatment activity performed by one or more research users, who collect, model and analyse research data, typically processed data, to extract insights that support the answer to some specific research question (i.e., conclusions).

DataInterpretationc

URI https://purls.helmholtz-metadaten.de/prima/dal#DataInterpretation
Description

Data interpretation is process or activity performed by one or more research users, who assign meaning to research data, typically analysed data, and determine the conclusions of the study, possibly in comparison with reference data.
Super-classes prov:Activityc

DataProcessingc

URI https://purls.helmholtz-metadaten.de/prima/dal#DataProcessing
Description

Data processing is a data treatment activity performed by one or more research users on research data, typically raw data, to prepare it for another step of the data analysis lifecycle, e.g., data analysis or data interpretation.
Super-classes prov:Activityc

ProcessedDatac

URI https://purls.helmholtz-metadaten.de/prima/dal#ProcessedData
Description

Processed data is research data which is the primary output of any kind of data processing or manipulation of raw data performed by one or more research users, possibly using research software, in order to prepare it for another step of the data analysis lifecycle, e.g., data analysis or data interpretation.
Super-classes core:ResearchDatac

RawDatac

URI https://purls.helmholtz-metadaten.de/prima/dal#RawData
Description

Raw data is research data which is the primary output of a measurement, collected by one or more research users using an instrument, before any subsequent data processing.
Super-classes core:ResearchDatac

ReferenceDatac

URI https://purls.helmholtz-metadaten.de/prima/dal#ReferenceData
Description

Research data not produced during the current study, which is used as reference to compare and/or to validate the outputs of the study, typically during the data analysis lifecycle.
Super-classes core:ResearchDatac

ResearchSoftwarec

URI https://purls.helmholtz-metadaten.de/prima/dal#ResearchSoftware
Description

Research software is software used to generate, process, analyse or access research data during any of the processes included in the data analysis lifecycle (possibly including data rendering, visualization, plotting).
