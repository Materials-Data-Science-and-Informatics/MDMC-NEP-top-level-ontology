Description

Data Processing c

IRI https://purls.helmholtz-metadaten.de/prima/dal#DataProcessing
Is Defined By https://purls.helmholtz-metadaten.de/skosmos/prima/dataprocessing
Description

Set of actions performed by one or more Research Users, possibly using a Technique and Research Software, on Research Data, typically Reference Data or Raw Data, to prepare it for one or more further processes, e.g., Model Preparation, computational Data Acquisition, and/or Data Analysis. Data Processing usually consists of routine actions. It may include: filtering, denoising, transformation, fusion and/or compression of Research Data, as well as calibration, normalization, statistical data reduction, background subtraction and/or correction of artifacts. The output of Data Processing is Processed Data.

Processed Data c

IRI https://purls.helmholtz-metadaten.de/prima/dataset#ProcessedData
Is Defined By https://purls.helmholtz-metadaten.de/skosmos/prima/processeddata
Description

Specific type of Research Data, primary output of any kind of Data Processing performed on Research Data (typically, but not limited to, Raw Data or Reference Data). Processed Data is usually an intermediate result, to be used as input of one or more further processes, e.g., Model Preparation or Data Analysis.
Sub Class Of core:ResearchData c
Restriction References ap some https://purls.helmholtz-metadaten.de/prima/dataset#ReferenceData c

Reference Data c

IRI https://purls.helmholtz-metadaten.de/prima/dataset#ReferenceData
Is Defined By https://purls.helmholtz-metadaten.de/skosmos/prima/referencedata
Description

Any Research Data from prior Studies or from literature, which is used in the course of a Study to, e.g., define a Model, compare Analyzed Data, and/or validate findings and results. Reference Data is typically retrieved from publicly accessible Data Repositories. Depending on the research context, Reference Data usually undergo some kind of Data Processing before being used.
